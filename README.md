# Sales Dashboard

An interactive sales analytics dashboard built using Python and Streamlit to analyze sales performance and generate business insights.

## Live Demo

🔗 Streamlit App:
https://sales-dashboard-mltjsbrwkllfrxaeflcyxq.streamlit.app/

## Project Overview

This project focuses on analyzing sales data and presenting key business metrics through an interactive dashboard.

The dashboard helps users understand:
- Revenue trends
- Sales performance
- Product/category analysis
- Business insights through visualizations

## Features

✅ Interactive sales dashboard  
✅ Data filtering options  
✅ KPI metrics display  
✅ Sales trend analysis  
✅ Product performance analysis  
✅ Data visualization charts  

## Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib / Plotly
- Git & GitHub
